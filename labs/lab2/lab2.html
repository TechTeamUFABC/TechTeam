Lab 2 - Imagem e Captura de Vídeo

Aquisição das imagens e vídeos

A fim de diminuir ao máximo a presença de sombras na imagem, foram utilizados 3 focos de luz utilizando a filmadora, sendo duas delas laterais e a outra frontal. Devido a problemas de bateria, as luzes laterais perderam a potência durante o experimento, de modo a permitir parcialmente o surgimento de sombras nas últimas imagens capturadas. Além disso, foi realizado o ajuste de temperatura de cor da filmadora para minimizar o pós-processamento

A configuração utilizando a câmera do celular foi mais simples, utilizando apenas uma luz de anel e as configurações de imagem já presentes no aplicativo de câmera.

Pós-processamento

O pós-processamento das imagens foi realizado utilizando o software Adobe Photoshop para as montagens, mudanças de cor e cortes, e a biblioteca OpenCV em Python para mudanças de resolução e cor.

  • Photoshop

    • Adobe Photoshop é um software caracterizado como editor de imagens bidimensionais do tipo raster (possuindo ainda algumas capacidades de edição típicas dos editores vectoriais) desenvolvido pela Adobe Systems. É considerado o líder no mercado dos editores de imagem profissionais, assim como o programa de facto para edição profissional de imagens digitais e trabalhos de pré-impressão.

    As fotografias dos integrantes do grupo foram tratadas utilizando o software Adobe Photoshop. O software também foi utilizado para manipulação e montagem com os avatares. Ele foi escolhido devido a sua vasta gama de recursos para edição e manipulação de imagens, e por um dos integrantes do grupo possuir uma licença do software.

  • OpenCV

    • O OpenCV é uma biblioteca do Python utilizada para visão computacional e Machine Learninig. Para este laboratório, utilizaremos o cv2, que é um dos módulo do OpenCV a fim de simplificar a instalação, já que não é necessário todas as funções do OpenCV.

    A principal função utilizada é a cv2.cvtColor(imagem, conversao), que converte a imagem passada como parâmetro para outro tipo de padrão de cor. Deste modo, utilizaremos essa função utilizando o parâmetro de conversão nativo do cv2.COLOR_BGR2GRAY, que converte do padrão RGB para cinza.

    O código para converter as imagens originais em preto e branco utilizando o cv2 encontra-se a seguir:

    
import cv2

# Import das imagens
walter = cv2.imread('C:\\Users\\Waltinho\\minami\\TechTeam\\labs\\lab2\imagens_editadas\\walter_100.jpg')
joao = cv2.imread('C:\\Users\\Waltinho\\minami\\TechTeam\\labs\\lab2\imagens_editadas\\joao_100.jpg')
adinan = cv2.imread('C:\\Users\\Waltinho\\minami\\TechTeam\\labs\\lab2\imagens_editadas\\adinan_100.jpg')
grupo = cv2.imread('C:\\Users\\Waltinho\\minami\\TechTeam\\labs\\lab2\imagens_editadas\\foto_grupo.jpg')

# Conversão
walter_gray = cv2.cvtColor(walter, cv2.COLOR_BGR2GRAY)
joao_gray = cv2.cvtColor(joao, cv2.COLOR_BGR2GRAY)
adinan_gray = cv2.cvtColor(adinan, cv2.COLOR_BGR2GRAY)
grupo_gray = cv2.cvtColor(grupo, cv2.COLOR_BGR2GRAY)

cv2.imwrite('walter_cinza.jpg',walter_gray)
cv2.imwrite('joao_cinza.jpg',joao_gray)
cv2.imwrite('adinan_cinza.jpg',adinan_gray)
cv2.imwrite('grupo_cinza.jpg',grupo_gray)
cv2.waitKey(0)
cv2.destroyAllWindows()
